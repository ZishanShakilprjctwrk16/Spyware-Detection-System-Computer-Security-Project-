# Integrated Spyware Detection and Investigation System

Windows 11 Security Tools Integration Prototype for Cybersecurity Investigation and Monitoring.

---

# Project Description

The **Integrated Spyware Detection and Investigation System** is a Python-based cybersecurity project developed for undergraduate Computer Security coursework.

The project integrates multiple open-source security investigation tools into a centralized Tkinter GUI dashboard to monitor suspicious activities related to spyware and malware behavior on Windows 11.

The system helps investigate:

- Open network ports
- Running processes
- Startup persistence
- Live network traffic
- Suspicious system behavior

This project demonstrates practical cybersecurity concepts including:
- network monitoring
- endpoint monitoring
- persistence analysis
- digital investigation
- security tool integration

---

# Features

## 1. Nmap Network Monitoring
Detects:
- open ports
- running services
- suspicious exposed services

---

## 2. OSQuery System Monitoring
Monitors:
- running processes
- suspicious process behavior
- executable paths

---

## 3. Wireshark Traffic Analysis
Captures:
- live network packets
- communication traffic
- suspicious outbound activity

---

## 4. Startup Persistence Investigation
Checks:
- Windows startup registry entries
- autorun persistence mechanisms

---

## 5. Investigation Report Generation
Generates:
- timestamped investigation reports
- scan logs
- monitoring outputs
